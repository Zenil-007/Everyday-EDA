🧬 Fertility Prediction Analysis (EDA + Machine Learning)

This project focuses on analyzing factors affecting fertility and building a predictive model to determine whether a pregnancy outcome will be successful or not.

📌 Problem Statement

Fertility is influenced by multiple biological and lifestyle factors such as age, BMI, stress, and medical conditions.

The objective of this project is to:

Perform Exploratory Data Analysis (EDA)
Identify key factors affecting fertility
Build a Logistic Regression model to predict pregnancy outcomes
📊 Dataset Overview

The dataset includes various features related to both male and female reproductive health.

🔑 Key Features:
Female Age
Male Age
BMI
Menstrual Regularity
PCOS (Polycystic Ovary Syndrome)
Stress Level
Smoking
Alcohol Intake
Sperm Count
🎯 Target Variable:
Pregnancy Outcome
1 → Successful
0 → Not Successful
🔍 Exploratory Data Analysis (EDA)

EDA was performed to understand the dataset and uncover patterns.

Key Steps:
Handling missing values
Checking data distributions
Identifying outliers
Analyzing correlations
Comparing categorical variables
📈 Insights:
Age plays a significant role in fertility outcomes
Higher stress levels negatively impact pregnancy success
PCOS is strongly associated with lower success rates
Lifestyle factors like smoking and alcohol intake influence fertility
